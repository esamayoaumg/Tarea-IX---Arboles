# Tarea-IX---Arboles

## Descripción
Este proyecto consiste en la implementación de un Árbol Binario de Búsqueda (BST) en C++, utilizando memoria dinámica.

El programa permite insertar valores y visualizar los recorridos del árbol:
- InOrden
- PreOrden
- PostOrden

Además, incluye un menú interactivo que permite al usuario realizar operaciones de forma continua.

## Funcionalidades
- Inserción de nodos en el árbol
- Recorrido InOrden (muestra los datos ordenados)
- Recorrido PreOrden
- Recorrido PostOrden
- Mostrar todos los recorridos
- Validación cuando el árbol está vacío
- Menú interactivo usando switch

## Estructura del proyecto

## Cómo ejecutar
1. Abrir el proyecto en CLion, CodeBlocks o cualquier compilador C++
2. Compilar el archivo `main.cpp`
3. Ejecutar el programa
4. Utilizar el menú para insertar datos y ver los recorridos

## Conceptos aplicados
- Árboles binarios de búsqueda (BST)
- Recursividad
- Punteros
- Estructuras dinámicas
- Control de flujo con switch

## Observaciones
El recorrido InOrden muestra los valores en orden ascendente, lo cual confirma que el árbol está correctamente implementado.
